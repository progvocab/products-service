Problem (Substring with Concatenation of All Words)
Given a string s and an array of words (same length), find all starting indices of substrings formed by concatenating all words exactly once.
Words can appear in any order.
Each word must be used exactly once.
No extra characters allowed in between.
Return all valid starting indices.


---

Solution Steps (Sliding Window + HashMap):

1. Store word frequency in a hashmap.


2. Let wordLen, numWords, windowSize = wordLen * numWords.


3. Iterate over wordLen starting offsets.


4. Use sliding window with left pointer.


5. Extract words of size wordLen.


6. Maintain current window frequency map.


7. If word count exceeds → shrink window.


8. If window matches required frequency → record index.


9. Continue sliding for all offsets.




---

Pattern:
Sliding Window + Hashing


---

Advantage over Brute Force:
Avoids checking all permutations.
Efficient O(n * wordLen).
Uses hashmap for quick frequency validation.
