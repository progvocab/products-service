Enterprise Search refers to the ability of organizations to **search, discover, and retrieve information** across their **internal data sources** (databases, file systems, emails, wikis, CRMs, intranets, etc.).

### Characteristics

* **Broad Data Coverage** â†’ Multiple data types: text, PDFs, spreadsheets, emails, images, logs.
* **Access Control** â†’ Respects user permissions (e.g., search results filtered by role).
* **Scalability** â†’ Works across millions or billions of documents.
* **Relevance Ranking** â†’ Prioritizes the most useful information.
* **Advanced Features** â†’ Faceted navigation, synonyms, spell-check, semantic search.

### Examples of Enterprise Search Tools

* **Amazon Kendra** (ML-powered search)
