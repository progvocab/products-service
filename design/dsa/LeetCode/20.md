Problem (Remove Nth Node From End of List)
Given a singly linked list, remove the nth node from the end.
Return the head of the modified list.
List size is at least 1.
You must do it in one pass.
n is always valid.


---

Solution Steps (Two Pointer):

1. Create a dummy node pointing to head.


2. Initialize two pointers: fast and slow at dummy.


3. Move fast n+1 steps ahead.


4. Move both fast and slow until fast reaches null.


5. Now slow is before the target node.


6. Update slow.next = slow.next.next.


7. This removes the nth node from end.


8. Return dummy.next.




---

Pattern:
Two Pointers (Fast & Slow)


---

Advantage over Brute Force:
Single pass → O(n).
No need to calculate length first.

Given a string containing ()[]{}, determine if it is valid.
A string is valid if brackets are closed in correct order.
Each opening bracket must have a matching closing bracket.
Closing must correspond to the most recent unmatched opening.
Return true if valid, else false.


---

Solution Steps (Stack):

1. Initialize an empty stack.


2. Create a map of closing → opening brackets.


3. Traverse each character in string.


4. If opening bracket → push to stack.


5. If closing bracket:


6. Check stack is not empty and top matches mapping.


7. If not → return false.


8. Else pop from stack.


9. After traversal, check stack is empty.


10. Return true if empty, else false.




---

Pattern:
Stack (LIFO matching)


---

Advantage over Brute Force:
Single pass → O(n).
Ensures correct nested order.
Avoids re-checking substrings.Problem (Merge Two Sorted Lists)
Given two sorted linked lists, merge them into one sorted list.
The result should be a single sorted linked list.
Reuse existing nodes (no extra list needed).
Return the head of the merged list.
Lists can be empty.


---

Solution Steps (Iterative Merge):

1. Create a dummy node and a pointer curr.


2. Compare heads of both lists.


3. Attach smaller node to curr.next.


4. Move that list pointer forward.


5. Move curr forward.


6. Repeat until one list is exhausted.


7. Attach remaining nodes of the other list.


8. Return dummy.next.




---

Pattern:
Two Pointers (Linked List Merge)


---

Advantage over Brute Force:
Single traversal → O(n + m).
No extra space for new nodes.
Efficient reuse of existing structure.
